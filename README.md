# isaacjmc.github.io
## Gracias por ver mi portafolioy y querer aprender de mi trabajo,

## A manera de resumen de este proyecto , consiste en una pagina web que se aloja de manera gratuita en github pages y 
## que consume datos de una hoja de calculo de google sheets para mostrarlos en la pagina web, cuyas peticiones se encuentra en un servidor gratuito en pythonanywhere.

## es decir que se puede actualizar los datos en un sheets y que se reflejen en la pagina web sin necesidad de actualizar el codigo y esto de manera
## gratuita para el usuario que quiera ver su portafolio. 

## Para desplegar el codigo se debe tener conocimiento de git y python, estoy trabajando en dejarlo lo mas facil de editar posible para 
## que cualquier persona pueda desplegarlo en su computadora y que pueda ser modificado para su uso personal.

## En este repositorio se encuentra el codigo fuente de la pagina web y el servidor que consume los datos de la hoja de calculo, tambien le pedi a una 
## IA que hiciera un instructivo facil para poder desplegarlo en github pages y pythonanywhere. Los archivos se encuentran en la carpeta 
## INSTRUCCIONES_FRONTEND.md y INSTRUCCIONES_BACKEND.md.
